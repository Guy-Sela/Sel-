About the Collection

Conceptual Timing is an evolving collection of clocks whose subject matter is the essence of time itself. The pieces are code based, and even the most technically far fetched clock in the collection — Universe Clock — is a real clock nonetheless. It will run the next moment, as long as it's still in tact when the next moment comes (86 billion years from now).

The conceptual clocks can appear anywhere there's a screen. They are set to be hypnotic, friendly to the contemplative mind and to the sublime ambiance seeking soul.
