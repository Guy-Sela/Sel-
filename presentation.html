Semi-Linear Clock is part of the Meditative Clocks collection by Code for Consciousness.

This clock always shows the right time, but never in the same way — "the next moment" is always a question mark, solved by an algorithm set to random.

Thus the notion that each moment is unique takes on a literal form.
